# TV APIs for Go

> Provides interfaces to different TV show information providers

## Installation
`go get` the API you're interested in:

    // The TV DB:
    $ go get github.com/rjelierse/go-tvapis/thetvdb
    // TV Maze:
    $ go get github.com/rjelierse/go-tvapis/tvmaze
